# 用数据看待世界——《魔鬼经济学》

# 为什么选择这本书

​ 《魔鬼经济学》是2021年读的第八本出版书籍。首先说明下为什么要选择这本书，有一段时间在B站上看浙江大学的《微观经济学》，课中的教授在视频中推荐的，那个时间点在微信读书上看了一会儿，后边忘记为啥停了，然后趁着电商平台4.23做活动入了实体书，可算是读完了。

# 书中部分例子

  1. 堕胎合法化引起美国犯罪率下降（选择堕胎的母亲，大部分认为自己不能够给孩子提供良好的成长环境，所以才会选择堕胎。而从恶劣家庭环境中长大的孩子犯罪率确实比正常家庭环境中长大的孩子高）
  2. 托儿所规定迟来接孩子要罚款，家长迟到率反而上升（书中用两个手段去衡量，托儿所使用经济手段替代了道道德手段，家长认为我既然要罚款了那么迟点去也就没事了，不会有心理负担）
  3. 教师作弊事件（这个案例的分析数据很强，建议去看原书）
  4. 决定工资多少的四个因素
    1. 有多少人愿意并且能够胜任
    2. 工作所需的专业技能
    3. 工作的不适程度
    4. 工作提供的服务的需求量
  5. 刻意自觉上三要素：
    1. 设定具体目标
    2. 立即获得反馈（有一说一，前两条游戏都中）
    3. 既要专心提高技能，又要注重结果
  6. 观察者效应：即悲剧发生时，如果有多个目击者，反而会导致目击者无动于衷；
  7. 人类的个体会因受到密切观察二产生很大变化（君子慎独）；
  8. 坦然承认自己不知道；
  9. 不要给自己定目标，或者定个大点的目标（例如给自己设定平板支撑60秒，五十多秒就感觉摇摇欲坠，如果不设定目标，一般都能坚持更久）
  10. 分辨想法的好坏并不容易，有一个窍门就是给想法一个冷静期，24小时后再去看这个想法；
  11. 说服别人可以试着讲故事。

# 个人想法

​ 这本书议案一开始一吸引我的就是数据分析的地方，从各种无关的数据之间挖掘事件的起因，也从各种数据之间找寻关联，能够得到神奇的结果。刚好昨天在看疑犯追踪第三季的第二季，讲述的一个数据公司负责人因泄露他人数据而招致报复的故事。或者说是这部剧的主线，从我目前的角度来看，所谓的机器就是拥有足够多的数据，从而去分析得出犯罪行为预兆。这一切无疑都是对数据的合理运用。

​ 在这几年，我们也常常听到一个词，那就是大数据。我也不知道所谓的大数据讲的究竟是什么，但是大数据却实实在在的影响我们的生活。衣食住行各个方面都有大数据的影响：美团算法从送餐数据分析，一步步压榨骑手的潜力（错误栗子）；正确例子：通过各大App的年度数据，分析自己到底是一个什么样的人（算法有待研究）。这篇文章，自己细细看来，好像说了很多，但是实际一想，好像又什么都没说。因为个人暂时也没有一双发现隐藏数据的眼睛，什么经验之谈啊什么都没有，等自己有发现了再来跟大家伙分享。
