# 为什么选择这本书

​ 应该是在搜索如何入门历史或者说能够比较系统性的去了解历史的时候看到别人推荐的，一直躺在购物车里。直到4.23世界读书日然后下单。这应该是2021的第七本书，也是第一本开始做笔记的书，为了让自己有个大概的印象，所以接下来可能是摘抄再加上部分个人瞎扯。

# 《中国通史》的知识点

# 中国历史分期

钱穆先生将中国历史分为五期：

  1. 史前史，有文字记载之前的历史，又可以将史前史分为两个时期：
    1. 旧石器时代：有使用石头工具，但是没有加工；
    2. 新石器时代：会对石头工具进行加工处理；
  2. 铜器时代：开始有文字记载
  3. 铁器时代
  4. 电器时代
  5. 原子能时代

# 商周春秋战国

  1. 周公是文王四子，武王的兄弟，成王的叔叔。武王原先要让位周公，周公不受。武王灭商二年后崩，成王只有十多岁，所以周公摄政；
  2. 春秋：一国一城，算是城市国家；
  3. 战国：一国多城，以军立国；
  4. 战国九雄：燕赵齐秦楚魏韩再加上宋（商）、中山；
  5. 合纵连横：
    1. 合纵：南北相连打击秦，苏秦
    2. 连横：东西联合服从秦，张仪
  6. 苏秦张仪都是鬼谷子的徒弟，鬼谷子的徒弟还有庞涓孙膑。

# 秦汉

  1. 秦是中国第一个统一政府，是贵族的；
  2. 汉是中国第一个士人政府，是平民的；
  3. 汉朝共二十六帝，享国四百多年。可能个人了解较多一点的就是汉高祖（秦末楚汉之争）、汉武帝（封狼居胥、罢黜百家，独尊儒术等）、光武帝（什么叫穿越者大战位面之子啊）以及灵帝献帝（汉末三国）

# 魏晋南北朝

  1. 五胡：氐、羌、鲜卑、羯以及匈奴；
  2. 匈奴俘虏晋帝，西晋南渡成东晋（神似南北宋）

# 隋唐

  1. 隋文帝（网上对他的评价很高，但是我个人没有找到关于隋文帝较好的书籍，有了解的烦请告知，十分感谢）代北周立国，隋炀帝奇观误国；
  2. 李世民：十八从军，二十四平天下，二十九称帝。
  3. 安史之乱不仅是唐由强转弱的节点，更是中国历史的转折；
  4. 唐亡有三大因素
    1. 藩镇割据
    2. 宦官专政
    3. 朋党作乱
  5. 唐后是五代，分别是：梁、唐、晋、汉、周——书中描述为中国历史最黑暗时期

# 宋元

  1. 宋继承自北周，太祖死弟继位，为太宗；
  2. 檀渊之盟：宋兄辽弟，宋每年送辽十万银，二十万绢；
  3. 靖康之耻：金虏宋徽宗、宋钦宗
  4. 宋有两次变法，一次是范仲淹的庆历变法，还有就是王安石的熙宁变法；
  5. 元朝是中国统治权首次落入异族手中；

# 明

  1. 明朝十六帝，享国二百七十七年；
  2. 明太祖多猜忌，杀尽功臣诸将
  3. 明初君王勤政，大臣清廉。嘉靖（强烈推荐《大明王朝1566》，讲的正是嘉靖时期）、隆庆之后，皇帝无心政事、宦官独揽大权、官僚不负责。

# 清

  1. 清代十帝，二百六十八年
  2. 清军事并不强，但是对手政治腐败，还有汉奸相助，所以得以代明；

# 最后

​ 个人感觉这本书与其叫中国通史，不如叫国史大纲会更加贴切。主要还是从大势方面描述中国历史从商周到晚清，书中有很多点都很有意思，但是没有细讲，需要自己通过其他渠道去拓展或者说了解相应的历史节点。
