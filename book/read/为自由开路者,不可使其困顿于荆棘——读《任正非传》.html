# 为自由开路者,不可使其困顿于荆棘——读《任正非传》

# 为什么选这本书

​ 2021年读完的第二本书籍，是孙力科的《任正非传》。出版时间是2017年4月，从2017年到现如今（2021年2月），华为经历了太多太多。也正是这期间所发生的一系列事情，让人不禁想去了解这个堪称伟大的公司。要了解这个公司，我的选择是从了解它的创始人开始，所以就选择了这本书——《任正非传》。

# 一些想法

​ 我个人认为，这本书出版名称虽然是《任正非传》，实际上也可以叫《华为传》（同作者在次年1月出版了《华为传》）。本书描述任正非的内容其实不多，更多的是从华为公司整体的角度，去描述任正非的一些行为，不过在我极其有限的个人传记阅读经历来看，是不可避免的。因为脱离环境去看待一个人的行为，是毫无意义的，正如同去年看的两本人物传记（《毛泽东传》以及《江泽明传》），如果脱离了中国共产党，两位领导核心的事似乎没有那么多可以讲的。

​ 在读《任正非传》中，有一本书时常出现，那就是《毛泽东选集》，书里也常常出现毛选中的名句，以及任正非根据对应语句做出的策略。例如：

农村包围城市。

​ 在攻克国内市场之后，任正非由香港开始，到俄罗斯，非洲以及拉美国家，再转战亚洲，最后啃下欧美。这步步为营，层层递进的战术一直到现在也特别好用。例如拼多多以及快手，一开始可能都是扎根于四五线城市，先啃下农村或者说落后地区作为基本盘，然后再冲击一二线。

要把我们的人搞得多多的，把敌人的人搞的少少的。

​ 华为两次进入美国市场的经历，就充分诠释了这句话。一开始的华为没有意识到这点，与其说是败给了思科，不如说是败给了媒体的舆论之中。在第一次失败后，华为团结一切可以团结的力量，例如司法联盟、民间团体以及思科的对手等等，终于在美国市场站稳了跟脚。

​ 案例不止以上两个，还有其他许多，就不在此过多赘述。

# 写在读后

​ 在《任正非传》最后一章最后一个小节“华为之父”中有这样一段话：

2016年初， 任正非在内部会议上针对华为的终端提出了更高的要求， 即5年后希望华为的终端收入能够 超过1000亿美元。 在过去的2015年， 这个数字尚且不到200亿美元。

​ 我在百度查了下，2019年华为终端收入超500亿美元（不太严谨，欢迎指正），离这个目标相差甚远，但是我们能够看到华为终端的成长：从“中华酷联”到2019年全球手机出货量全球第二，仅次于三星，华为所依仗的应当不仅仅是海思。

​ 让我真正有想去去了解华为的契机是美国对华为的制裁，我就在想一家企业何德何能能够让全球霸主脸都不要了，去针对一家企业。也“得益于”美国的制裁，2020年全球手机出货量华为排在第三，同比增长-18%。

​ 我想，大部分中国人心中可能都有一种英雄主义色彩，十分佩服或者说敬仰那些“知其不可而为之”的人。从古至今数不胜数，例如屈原、戚继光、诸葛亮以及文天祥等等等等，尤其是这种带有悲壮主义的英雄，就更加想让人能够看到他们的美好结局，所以有那么多魂穿三国、晚明或者说宋末的小说。历史不可改变，但是对于现在的华为，我们可以做一些力所能及的事情，正如毛选中的一句话：

凡是敌人反对的，我们就要拥护；凡是敌人拥护的，我们就要反对。

​ 华为之所以被制裁，是因为威胁到了西方国家的部分科技的领先地位。同时也正是因为这些领先科技，西方国家才能在保持低工作时间以及高福利的情况（想起之前在网上跟别人聊过，他很讨厌华为，因为996，羡慕并希望我国的工作时间能够照西方的来；我认为华为的996是无可避免的，因为华为作为一个追赶者，只能付出比领先者更多的努力，才能够拉平差距甚至追上他们）。所以无论是我自己购买或者给别人推荐产品，在符合条件的产品中，我更加倾向于推荐华为或者说国产的产品，毕竟市场是用钱投票的。

​ 匆匆写的，比较乱，可能有很多地方不连贯或者说语句不通畅，还需要个人多多练习，感谢您能抽空看这样一些文字，并看到这里。

为众人抱火者，不可使其冻毙于风雪；为自由开路者，不可使其困顿于荆棘。

​ 最后，如果觉得文中的部分观点与您的观点有所出入，那么，您的观点是对的。
