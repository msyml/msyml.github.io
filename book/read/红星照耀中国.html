# 信仰的力量——《红星照耀中国》

# 为什么选择这本书

​ 2021年读的第五本书，是美国记者埃德加·斯诺所著，董乐山所译的《红星照耀中国》（又名《西行漫记》）。因为书是去年买的，已经忘记当时为什么要买这本书了，应该是说能够真实的了解在抗战时期真实中国农村的面貌，所以才去购买的。但实际上本书所描写的事情基本都发生在西安事变之前（也就是民族抗日统一战线形成之前）。

# 主要记忆

	这本书主要是讲作者作为一个外国记者，从国军统治区到达苏区统治区，在路上的所见所闻所想以及最后对西安事变的一些描述和想法。采访了红军一系列领导人以及部分红军战士的记录。书中大概的介绍了贺龙、毛泽东、彭德怀以及朱德的生平，描述了红军在各地的各种事迹以及到达苏区之前的长征。

​ 其他的一切略过不说，书中让我印象最深刻也是最打动我的，是红军战士或者说那个时代共产党人的精神。为国为家能够做到无私忘我，并且流露出一股天不怕地不怕的精神，或者说是能够感染他人的精神，吸引更多志同道合的的同志，一起为了共产主义事业而奋斗。希望自己也能够找到让自己为之奋斗终身的事业，向前辈学习。不忘初心，方能始终。

# 最后

​ 最后也不知道该咋说，用教员的几首诗词激励下。

《七律·长征》

红军不怕远征难，万水千山只等闲。

五岭逶迤腾细浪，乌蒙磅礴走泥丸。

金沙水拍云崖暖，大渡桥横铁索寒。

更喜岷山千里雪，三军过后尽开颜。

《忆秦娥·娄山关》

西风烈，长空雁叫霜晨月。

霜晨月，马蹄声碎，喇叭声咽。

雄关漫道真如铁，而今迈步从头越。

从头越，苍山如海，残阳如血。

《七律·人民解放军占领南京》

钟山风雨起苍黄，百万雄师过大江。

虎踞龙盘今胜昔，天翻地覆慨而慷。

宜将剩勇追穷寇，不可沽名学霸王。

天若有情天亦老，人间正道是沧桑。
