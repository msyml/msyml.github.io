# 为什么选这本书

​ 2021年的第四本书，是博多·舍费尔的《小狗钱钱》以及《小狗钱钱2》。为什么选择这本书呢，其一是因为好友的推荐，其二是因为自己有在进行一些理财的东西，所以想要了解一些有关于理财的知识。

# 大致内容

​ 第一本主要有关于理财，故事从主人公吉娅捡到一只会说话的狗开始，主人公给狗取名叫钱钱。然后在钱钱以及钱钱的前主人金先生的帮助下，主人公不仅实现了自己的梦想，还帮助家庭摆脱了债务危机。第二本则是讲述主人公成长的故事，所谓衣食足而知荣辱，仓廪实而知礼节。

# 梦想储蓄罐以及梦想相册

​ 像是我们个人制定目标一样，设定好一个目标以及完成期限，然后从自己的每笔收入之中抽取一部分去存入梦想储蓄罐。所谓积水成渊，蛟龙生焉，一个道理。

# 合理分配收入以及支出

​ 如果有了固定收入之后，可以给自己的固定收入设定一个比例，例如书中分配方法：

  1. 50%：用于存储升值，变成一个会下蛋的鹅；
  2. 40%：存储到梦想存储罐中，实现短期梦想；
  3. 10%：用于支付生活必须支出。

应该也要给自己设定一个合理的比例。其次就是支出，在任何花费之前，都想想这一笔支出到底有没有必要，这样大概率可以减少很多非必要支出。

# 72以及翻倍

​ 假如一种投资年收益率有12%，那么你资产需要翻倍的时间为72/12 = 6年。同样的，物价要翻倍也可以这么算，就例如通货膨胀率为3%，那么物价翻倍所需要的时间为72/3 = 24年。

# 养成优秀品格的七条准则

  1. 友好亲和
  2. 勇于承担
  3. 善待他人
  4. 帮助给予
  5. 感恩之心
  6. 勤学不辍（chuò）
  7. 值得信赖

要养成这些习惯可能需要终生去锻炼，所以书中推荐每天去实行一种，直到掌握。

# 收尾

​ 这两本书可以说是给一些较为迷茫的人指明人生最终需要达到的目标或者说是方向。作者在第二本书中，使用甜甜圈做了一个比喻，将充裕的物质比作甜甜圈外边那个圈，优秀的品格比作甜甜圈那个看不见的内圈，虽然看不见，但是如果没有内部，那么甜甜圈也不是甜甜圈了。只能说
