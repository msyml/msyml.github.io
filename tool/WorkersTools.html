# 打工人常用工具推荐

君子性非异也，善假于物也 ——《荀子》

身为一个新时代打工人，高级民工，日常工作差不多100%的时间在跟电脑打交道。为了提高效率，争取多看点自己感兴趣的东西创造更多价值，免不了需要一些奇巧工具来帮忙，下边列举一些自己经常使用的工具，希望对看了文章的你能够有所帮助，也算是抛砖引玉，你如果有好工具也可以留言分享下鸭~

# uToolsopen in new window

日常使用第一多的工具（集），移动端类似的有“一个木函”之类的东西。个人日常用的比较多的插件有Json编辑器、讯飞OCR、剪切板、自动化助手、随机生成假数据、JS文档、CSS文档及计算稿纸等等等等。当然这个工具也有开发API，支持自己做定制化开发。下边随便举几个常用工具例子：

  • 二维码助手：支持将文本转化为指定二维码，支持解析二维码获取文本（对我来说基本可以替代草料二维码）
  • 讯飞OCR：继承科大讯飞的OCR接口，可以将图片中的文字转化为文本，方便复制
  • 聚合翻译：继承了有道、百度以及Google翻译（个人一般用来定义变量名）
  • 剪切板：现在大多数输入法可能也继承了相关功能，但是这个还是比较好用的，可以根据类型筛选，好久之前的记录也能保存住

  • 类似其他的还有文件批量重命名、Todo、备忘录等等

这个工具个人认为是个宝藏，还有很多插件等待发掘~

# Snipasteopen in new window

这是一个截图工具~大部分功能跟微信、QQ的截图功能差不多，用的最多的是类型图钉一样的功能，截图后可以把图片戳在屏幕上，然后滑动滚轮可以控制大小。就类似下图这样戳起来，对于单屏党非常实用。

# VsCodeopen in new window

神中神，个人心中天下第一文本编辑器，通过各种奇奇怪怪的插件，基本可以实现查看任意格式文件。
