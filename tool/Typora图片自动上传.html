# 使用PicGo搭配七牛云，完成Typora图片自动上传

# 0. 必备工具

  1. 七牛云对象存储（10GB免费存储空间，http流量十万次请求免费）
  2. 域名一个
  3. PicGo

# 1. 申请七牛云

​ 登录七牛云网站open in new window，填写注册信息。手机号注册完成后需要绑定邮箱，才可以进行实名认证。实名认证完成后，即可开通对应存储空间服务。设置好存储空间名称以及存储区域即可完成创建。

​ 完成创建后，点击右上角头像 -> 密钥管理，即可获取对应密钥。

​ 其次还需要一个域名，这个就不在这里展开了。有了域名之后，点击对象存储 -> 空间管理 -> 域名管理，去绑定对应域名，然后以个人所使用的腾讯云的dnsPod为例子

这样就算是完成了七牛云的绑定。

# 2. 配置PicGo以及Typora

	打开Typora，选择文件 -> 偏好设置 -> 图像

​ 接着打开PicGo，图床设置，选择七牛云，填上对应的信息，尤其注意确认存储区域，对应关系如下

华东华北华南北美东南亚
z0z1z2z3z4

完成后点击确定，然后设为默认图床。设置完成后可以点击到PicGo设置中，配置一些个人偏爱设置。

然后就可以复制一张图片，粘贴到Typora中，就会自动上传到七牛云存储中了。
