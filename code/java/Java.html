  4. MP自带策略（雪花策略）

IdType:

  1. Auto：自动增长
  2. ID_WORKER：MP自带策略，生成一个19位的值，数值类型使用
  3. ID_WORKER_STR：MP自带策略，生成19位的值，字符串类型使用
  4. INPUT：需要自己设置
  5. NONE：不用任何设置
  6. UUID：随机唯一值
