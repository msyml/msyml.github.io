# 网格

和 Canvas 用的差不多，以页面左上角为(0,0)坐标点，坐标以像素为单位，x 轴正方向是向右，y 轴正方向是向下。

# 像素

基本上，一个像素对应输出设备上的一个像素。SVG 可以定义绝对大小，同时 SVG 也能使用相对大小，只需给出数字。

<svg width="100" height="100"></svg>
1

上面的元素定义了一个100*100的SVG画布，这里1用户单位等于1屏幕单位。

<svg width="200" height="200" viewBox="0 0 100 100"></svg>
1

这里画布定义的尺寸是200x200px。但是，viewBox属性定义了画布上可以显示的区域：从(0,0)点开始，100宽x100高的区域。这个1002的区域，会放到2002的画布上展示，于是就形成了放大两倍的效果。

类似这样我们也可以推出缩放的效果，例如画布大小为100，viewBox为200，就相当于缩小了。

# 矩形<rect>

<rect x="10" y="10" width="30" height="30"/>
<rect x="60" y="10" rx="10" ry="10" width="30" height="30"/>
1
2
  • x：矩形左上角的x位置
  • y：矩形左上角的y位置
  • width：矩形的宽度
  • height：矩形的高度
  • rx：圆角的x方位半径
  • ry：圆角的y方位半半径

# 圆形<circle>

<circle cx="25" cy="75" r="20"/>
1
  • r：元的半径
  • cx：圆心的x坐标
  • cy：圆心的y坐标

# 椭圆<ellipse>

<ellipse cx="75" cy="75" rx="20" ry="5"/>
1
  • rx：椭圆的x半径
  • ry：椭圆的y半径
  • cx：椭圆中心的x坐标
  • cy：椭圆中心的y坐标

# 线条<line>

<line x1="10" x2="50" y1="110" y2="150"/>
1
  • x1：起点的x坐标
  • y1：起点的y坐标
  • x2：终点的x坐标
  • y2：终点的y坐标

# 折线<polyline>

<polyline points="60 110, 65 120, 70 115, 75 130, 80 125, 85 140, 90 135, 95 150, 100 145"/>
1
  • points：点集数列。每个数字使用空白、都好、终止命令符或者换行符分隔。每个点必须包含两个数字（x坐标以及y坐标），类似上边的折线可以表示为(60,110)-(65,120)-(70,115)...

# 多边形<polygon>

多边形跟折线类似，不同的是，多边形的路径在最后一个点处自动回到第一个点。

<polygon points="50 160, 55 180, 70 180, 60 190, 65 205, 50 195, 35 205, 40 190, 30 180, 45 180"/>
1
  • points：点集数列。每个数字用空白符、逗号、终止命令或者换行符分隔开。每个点必须包含2个数字，一个是x坐标，一个是y坐标。所以点列表 (0,0), (1,1) 和(2,2)可以写成这样：“0 0, 1 1, 2 2”。路径绘制完后闭合图形，所以最终的直线将从位置(2,2)连接到位置(0,0)。

# 路径<path>

<path d="M 20 230 Q 40 205, 50 230 T 90230"/>
1
  • d：一个点集数列以及其他关于如何绘制路径的信息。详情参照**Pathsopen in new window**了解更多。
