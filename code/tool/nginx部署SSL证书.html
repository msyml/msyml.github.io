# 下载证书

首先下载对应证书，然后解压，进入nginx文件夹得到1_hostname.com_bundle.crt（证书文件）以及2_hostname.com.key（私钥文件）两个文件。然后通过仓库或者xftp之类的工具，将这两个文件复制到nginx所部署的服务器上。

# 复制

// 复制个人主页打包后文件到镜像
docker cp  /data/person-web-site/dist/. nginx:/usr/share/nginx/html/person/
// 复制nginx配置
docker cp /data/nginx/default.conf 3c8dee24b288:/etc/nginx/conf.d/default.conf
// 复制证书
docker cp /data/note/nginx_config/1_msyml.cn_bundle.crt 3c8dee24b288:/etc/nginx/1_msyml.cn_bundle.crt
docker cp /data/note/nginx_config/2_msyml.cn.key 3c8dee24b288:/etc/nginx/2_msyml.cn.key

# 启动docker容器

-v的作用分别是：

  1. 复制nginx配置
  2. 复制证书
  3. 复制私钥
  4. 复制nginx service配置
  5. 复制react站点打包文件
  6. 复制vue站点打包文件
docker run --name nginx -p 443:443 -p 80:80 -v /data/note/nginx_config/nginx.conf:/etc/nginx/nginx.conf -v /data/note/nginx_config/1_msyml.cn_bundle.crt:/etc/nginx/1_msyml.cn_bundle.crt -v /data/note/nginx_config/2_msyml.cn.key:/etc/nginx/2_msyml.cn.key -v /data/note/nginx_config/1_api.msyml.cn_bundle.crt:/etc/nginx/1_api.msyml.cn_bundle.crt -v /data/note/nginx_config/2_api.msyml.cn.key:/etc/nginx/2_api.msyml.cn.key -v /data/note/nginx_config/1_shop.msyml.cn_bundle.crt:/etc/nginx/1_shop.msyml.cn_bundle.crt -v /data/note/nginx_config/2_shop.msyml.cn.key:/etc/nginx/2_shop.msyml.cn.key -v /data/note/nginx_config/default.conf:/etc/nginx/conf.d/default.conf -v /data/note/docs/.vuepress/dist/.:/usr/share/nginx/html/person/ -v /data/shop/dist/.:/usr/share/nginx/html/shop/ -d nginx
