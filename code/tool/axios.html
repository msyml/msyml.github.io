# axios源码

# 文件结构

  • dist 项目输出目录
  • lib 项目源码目录
    • adapters 定义请求的适配器xhr、http
      • http.js 实现http适配器
      • xhr.js 实现xhr适配器
    • cancel 定义取消功能
    • core 一些核心功能
      • axios axios的核心主类，构造函数文件
      • buildFullPath 构建完整url的函数文件
      • createError 创建Error对象
      • dispatchRequest 调用某个适配器发送请求
      • enhanceError 更新错误对象
      • interceptorManager 拦截器管理器对象构造函数
      • mergeConfig 合并配置
      • settle 根据响应状态码改变Promise对象状态
      • transformData 对请求结果做格式化
    • helpers 功能函数
    • axios 入口文件
    • defaults axios默认配置文件
    • utils 工具函数文件
