# Vue

# 开始之前

一套用于构建用户界面的渐进式JavaScript框架。

  1. 采用组件化模式，提高代码复用率、让代码更好维护

  3. 使用虚拟DOM+优秀的Diff算法，尽量复用DOM

# 数据代理

Object.defineproperty方法，给对象添加属性

let person = {
    name: "张三"，
    sex: "man",
}

let number = 18

// 这样添加的数据，不可以枚举，不参与遍历
// 值不可以修改
Object.defineProperty(person,'age',{
    value: 18,
    ebynerable: true, // 控制属性是否可以被枚举，默认值false
    writeable: true, // 控制属性是否可以被修改，默认值false
    configurable: true, // 控制属性是否可以被删除，默认值false
    // 当有人读取person的age属性时，get函数就会被调用，切返回值就是age的值
    get(){
    	// 这样如果number改变，person.age也会跟着改变 
    	return number;
	}，
    // 当有人修改age的属性时，set函数就会被调用，且会受到修改的具体值
    set(value){
    	number = value;
	},
})

// 利用get以及set就可以实现类似Vue的双向绑定

// 删除对象中的某个值，类似下
delete person.name

// 获取对象中所有属性名的值
Object.keys(person);// ["name","sex"]

console.log(person);// {name: "张三",sex: "man", age: 18}

# 事件处理

<!-- .prevent：阻止默认事件 -->
<a href="https://www.baidu.com" @click.prevent="showInfo">点我弹窗</a>
<!-- .stop：阻止事件冒泡 -->
<div @click="showInfo">
    <button @click.stop="showInfo">
        点我提示
    </button>
</div>
<!-- .once：事件只会触发一次 -->
<button @click.once="showInfo">
    点我提示
</button>
<!-- .capture：使用事件的捕获模式: -->
<!-- 捕获由外向内，冒泡由内向外,在捕获阶段处理事件。外部的会优先处理，内部后处理 -->
<div @click.capture="showInfo" class="box1">
    div1
    <div @click="showInfo" class="box2">
        div2
    </div>
</div>
<!-- .self：只有event.target是当前操作元素时才触发事件 -->
<div @click.self="showInfo">
    <button @click.stop="showInfo">
        点我提示
    </button>
</div>
<!-- .passive：事件的默认行为立即执行，无需等待事件回调执行完毕 -->
<!-- scroll给滚动条加上的滚动事件: 到顶部或者底部后不再触发 -->
<!-- wheel鼠标的滚轮发生滚动触发事件：只要鼠标滚轮还在动，就一直触发 -->
<!-- wheel事件如果加上passive修饰符，不用等循环输出完成，直接执行默认行为（上下滚动）。如果不加passive则要等循环先输出完成后，才会执行滚动。 -->
<ul @scroll="scroll" @wheel="scroll" class="list">
    <li>1</li>
    <li>2</li>
    <li>3</li>
    <li>4</li>
</ul>

new Vue({
    methods:{
        scroll(){

                console.log(i);
            }
            console.log("1")
        }

# 监测数组

通过数组下标直接修改数组中的值，并不能监测到。可以通过下列7个方法更改数组，Vue才能监测到：

  • push：往最后加值
  • pop：删除最后一项
  • shift：删除第一项
  • unshift：添加第一项
  • splice：指定位置 替换/插入/删除
  • sort：排序
  • reverse：颠倒数组

Vue重写了这几个方法，调用的push不是Array原型上的方法了，而是Vue重写的。

首先调用一次原先的方法，其次是根据数组变化操作虚拟DOM。

也可以通过Vue.set更改(array, index, value)

# 总结

  1. Vue会监视Data中所有层次的数据
  2. 如何监视对象中的数据？
    1. 通过setter实现监视
      1. 对象中后追加的属性，Vue默认不做响应式处理
      2. 如需给后添加的属性做响应式，需使用如下API
        1. Vue.set || vm.$set
  3. 如何监测数组中的数据
    1. 通过包裹数组更新元素的方法实现，本质就是做了两件事
      1. 调用原生的对应的方法对数组进行更新
      2. 重新解析模版，进而更新页面
  4. 在Vue修改数组中的元素一定要用如下方法
    1. push、pop、shift、unshift、splice、sort、reverse
    2. Vue.set或者vm.$set

# setup

所有组合API的“表演的舞台”。

组件中所用到的：数据、方法等等，均要配置在setup中。

两种返回值：

  1. 若返回一个对象，则对象中的属性、方法可以直接在模版中使用
  2. 若返回一个渲染函数：则可以自定义渲染内容（了解）

setup不能是async函数

<template>
{{name}}
{{age}}
</template>
<script>
    import {h} from 'vue'
export default{
    name: "APP",
    // 此处只是测试setup,暂时不考虑响应式
    setup(){
        let name = "name";
        let age = 18;
        function sayHello(){
            alert(`我叫${name},我${age}岁了，你好！`)
        }
        //return {
        //    name,
        //    age,
        //    sayHello,
        //}
        return () => {
            return h('h1','出')
        }
    }
}
</script>
