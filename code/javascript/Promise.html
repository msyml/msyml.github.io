# Promise介绍与基本使用

Promise是JS进行异步编程的新解决方案。

# 为什么使用Promise

  • 旧的：必须在启动异步任务前指定
  • Promise：启动异步任务 -> 返回Promise对象 -> 给promise对象绑定回调函数。
  • 支持链式调用，可以解决回调地狱问题。

借助until.promisify将(err, res)风格函数进行promise转化。

# 实现抽奖程序

btn.addEventListener('click',function(){
    const p = new Promise((resolve, reject) => {
        setTimeout(() => {
            let n = rand(1,100);
            if(n <= 30){
                resolve();
            }else{
                reject();
            }
        },1000)
    })
    
    p.then((res) => {},(err) => {})
})

# Promise API

# Promise.resolve

let p1 = Promise.resolve(111);
// 如果传入的参数为非Promise类型的对象，则返回的结果为成功promise对象
// 如果传入的参数为Promise对象，则参数的结果决定了resolve的结果
p1.then(res => console.log(res)); // 111

# Promise.all

let p1 = new Promise((resolve,reject) => resolve('ok'));
let p2 = new Promise((resolve,reject) => resolve('ok'));
let p3 = new Promise((resolve,reject) => resolve('ok'));
let p4 = new Promise((resolve,reject) => resolve('ok'));

const result = Promise.all([p1,p2,p3]);// 只有数组内部Promise全成功，这个方法返回值才是成功的
// 返回的结果是一个数组，是各个Promise对象的结果。
// 如果有失败的Promise，那个结果就是失败的那个Promise。
